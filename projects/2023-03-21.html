재택사용자 관리 웹 사이트

개발기간

2022-02-30 - 2023-11-31

개발환경

VPN, SpringBoot, DebianOS, Java, Mariadb, PaceMaker, React

요약

정부 과제 사업으로 공공기관내의 사용자PC들을 재택에서도
업무환경과 동일하게 사용할 수 있도록 휴대용 노트북을 특정 서버에 등록하여
선행인증 VPN을 통한 재택 업무환경을 제공할 수 있는 사이트 개발 프로젝트입니다
사용자들의 접속현황을 알 수 있고 사용자PC의 보안상태도 확인이 가능합니다

기능

(1)tcp
(2)네트워크

맡은역할

(1) 서버와 PC간의 통신 모듈 개발
(2) 서버 API 개발
(3) 프로세스 감시 Agent 개발
